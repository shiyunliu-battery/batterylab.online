Tests at module level
In the battery test standards some tests exist that are not described in this document. These tests
are not possible at cell level.
These tests are:
– Roll-over
– Thermal control check
– Propagation of thermal runaway
– Rapid charging and discharging
– Imbalanced Charge
– Overcharge voltage control check
– Overcharge current control check
– Over-discharge current control check
– Humidity
– Dewing
– Immersion
– Salt spray
– Rain test
– Electromagnetic susceptibility
